# FUTURE_CS_03

## API Security Risk Analysis

### Objective
Analyze a public API and identify potential security risks.

### Tools Used
- Browser DevTools
- JSONPlaceholder API
- Canva
- GitHub

### Findings
- No Authentication Required
- User Data Exposure
- Missing Rate Limiting
- Input Validation Concerns

### Recommendations
- Implement Authentication
- Restrict Data Exposure
- Apply Rate Limiting
- Improve Input Validation

### Deliverables
- API Security Risk Analysis Report
- Screenshots
- Documentation
